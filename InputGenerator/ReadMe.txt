Для использования утилиты генерации тестов необходимо скомпилировать файл InputGenerator.cpp.
Использование утилиты поддерживает 2 опции:
1) InputGenerator.exe КоличествоДронов КоличествоЦелей
Генерирует случайное ограничение на дальность полёта, случайный список точек и выводит все эти данные в консоль.
2) InputGenerator.exe КоличествоДронов КоличествоЦелей НомерФайла
Делает то же самое, но записывает результат в файл "НомерФайла.txt"
